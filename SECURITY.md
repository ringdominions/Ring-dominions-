# Security Policy

## Supported Versions

Ring Dominion is a single-file, client-side game. The latest version on the `main` branch is the only actively supported version.

| Version | Supported |
| ------- | --------- |
| main    | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability (e.g. an XSS vector, unsafe use of `eval`/`innerHTML`, or a wallet-connection related issue), please report it privately rather than opening a public issue:

1. Open a [GitHub Security Advisory](../../security/advisories/new) for this repository, **or**
2. Contact the maintainers directly via [X/Twitter @Devringdominion](https://x.com/Devringdominion).

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We'll do our best to respond promptly and keep you updated as the issue is addressed.
